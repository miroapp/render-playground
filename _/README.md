# Rendering Playground

Playground for Computer Graphics experiments and benchmarking using GPU acceleration APIs (WebGL / WebGPU / etc.)

## Development

Install and update dependencies

```
yarn install
```

Run development server

```
yarn start
```

## WebGL experiments

- [x] Simple batching
- [x] Instancing
- [x] Shader branching

## Tech. Stack

- Bundler: Parcel JS
